# 📝 Simple Todo List Web App

A clean and minimal Todo List web application built using **HTML, CSS, and JavaScript (no frameworks)**.

This app allows users to add tasks, mark them as completed, delete them, and automatically store tasks using the browser's Local Storage.

---

## 🚀 Features

- ✅ Add new tasks
- ✔ Mark tasks as completed
- 🗑 Delete tasks
- 💾 Persistent storage using LocalStorage
- 🔄 Tasks remain after page refresh
- 🎨 Clean and responsive UI
- ⚡ Lightweight (no frameworks)

---

## 🛠 Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- Browser LocalStorage API
